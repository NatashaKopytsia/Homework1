Курс верстки HTML/CSS

расписания занятий

  1. Вводное занятие. Знакомство с CSS и HTML, основные принципы их функционирования. HTML тэги и CSS селекторы.
  2. Блочная модель. Свойство display. Фоновые изображения.
  3. CSS: управление расположением элемента на странице (position, float). HTML: ознакомление с наборами тэгов используемых для верстки таблиц и форм.
  4. Краткий обзор текстовых редакторов используемых в верстке. Прицельный обзор редактора Sublime Text и его плагинов.
  5. Базовые навыки работы с графически редактором Adobe Photoshop.
  6. Спрайты. Псевдоклассы. Псевдоэлементы.
  7. Возможности CSS в решении стандартных задач визуального оформления (тени, градиенты, скргуление углов, фильтры, прочее).
  8. Единицы измерения в CSS. "Фиксированные" и "резиновые" макеты.
  9. Практическое занятие. Верстка макета.
  10. Вводное занятие по JavaScript.
  11. Знакомство с jQuery. Подключение готовых решений на примере скрипта слайдера. Обзор частоиспользуемых jQuery плагинов.
  12. Библиотека jQuery UI.
  13. Написание своих несложных сценариев на js (аккордеон, поп-ап, навигация по странице, прочее).
  14. Практическое занятие по js.
  15. Адаптивная верстка. Медиа-запросы.
  16. Адаптивная верстка. Медиа-запросы. (практическое занятие).
  17. Флекс-бокс (теоретическое занятие)
  18. Флекс-бокс (практическое занятие, решение задач).
  19. Верстка писем-рассылок.
  20. CSS анимация. Transition. Transform.
  21. Bootstrap.
  22. HTML и CSS препроцессоры.
  23. Закрепление пройденного материала. Оптимизация рабочего процесса.
  24. Заключительное занятие. Перспективы дальнейшего развития. Обзор смежных специальностей. Обзор фриланс-бирж.
